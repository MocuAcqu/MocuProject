## 英文寫作分析器
這是來自資料結構課程的期末專題，我串接大型語言模型 (Gemini-1.5-Flash)，藉由下達指令讓他生成明確的分析結果內容，並藉由資料庫 (supabase) 儲存使用者帳號與分析結果的歷史紀錄。在前端還結合 Quill 讓使用者可以編輯生成結果，下載成修改版本的 PDF 檔案。

## Tina的人生模擬器
